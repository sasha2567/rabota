Сайт тестирования.

Для конфигурации БД необходимо внести изменения в файл config.php

Главный скрипт - index.php

Все файлы контроллеров в папке core
В соответствии с названием файл-контроллер выполняет свои действия

Все файлы моделей данных в папке models

Все файлы страниц в папке pages

Структура работы сайта - движок MVC типа.
Главный файл подключает определенный контроллер (в зависимости от строки запроса), 
контроллер используя модель подгружает данные из БД и передает их на страницу. 

На сайте используется технология сессий для проведения авторизации и работы на сайте.
Так же предусмотрена регистрация.
